
Rewards Application

**Tech Stack**
- Java 8
- Spring Boot
- H2

**Run**
mvn spring-boot:run

**API**
GET /api/v1/rewards/{customerId}?startDate=YYYY-MM-DD&endDate=YYYY-MM-DD
GET /api/v1/rewards

**Features**
- Reward calculation
- Monthly aggregation
- DTO
- Validation + Exception Handling

**Future Improvements**
- Pagination
- Async processing
